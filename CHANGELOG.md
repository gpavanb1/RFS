# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [0.2.0] - 2026-02-27
### Added
- FPV table visualization and Post-processing script (`post_process.py`) for reaction progress rate analysis
- Finite difference implementation for derivative computation (`df_dlmbda`)
- S-curve benchmarking and output capabilities
- Enhanced output file organization and management

### Changed
- Updated constructor parameters and weighted norms for improved robustness
- Internal step counting for flamelet problem tracking
- Fixed tabulation keys for better data consistency

### Fixed
- Various tabulation and data handling improvements

## [0.1.0] - 2024-10-20
### Added
- Moved functionality entirely inside `FlameletProblem`

### Fixed
- Updated dependencies and added support for Cantera 3.0
